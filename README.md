# 포트폴리오 템플릿

---

## 시작 가이드

### 요구 사항

- Node.js v22.13.0 (LTS)
- nvm v0.40.1 이상
- pnpm v9

### 설치 및 실행

```bash
// package
npm install

// run
npm run dev
```
